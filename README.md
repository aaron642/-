# 動物人格識人系統 App 試作版

這是一個 Android 原生 App 專案，可用 GitHub Actions 自動打包成 APK。

## 功能
- 12 題動物人格測驗
- 6 種人格：老鷹型、獅子型、海豚型、貓頭鷹型、鹿型、狼型
- 結果頁：人格解析、溝通建議、帶領提醒
- 支援重新測驗

## 上傳 GitHub 後產生 APK
1. 在 GitHub 建立新的 Repository。
2. 上傳本專案所有檔案。
3. 進入 Actions。
4. 點選 Build Android APK。
5. 按 Run workflow，或上傳後等待自動執行。
6. 成功後到 Artifacts 下載 animal-personality-debug-apk。
7. 解壓縮後可取得 app-debug.apk。

## 注意
這是試作版，適合先測試概念。若要正式上架或商用，建議再補：美術首頁、品牌 Logo、完整測驗邏輯、隱私權政策、正式簽章 APK。
